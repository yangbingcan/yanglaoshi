# Trae IDE Auto Growth Skill Manager

自动成长技能管理器，沉淀对话经验并优化技能。

## 触发条件

- 用户对技能输出给出反馈
- 用户想要创建/迭代技能
- 用户需要从对话中捕获经验

## 核心功能

### 1. 经验捕获
- 用户偏好记录
- 踩坑经验沉淀
- 最佳实践总结
- 决策逻辑记录

### 2. 技能迭代
- 根据反馈优化
- 添加新规则
- 更新工作流程
- 完善输出格式

### 3. 知识管理
- 分类存储
- 版本追踪
- 快速检索
- 跨项目复用

## 工作流程

```
对话发生 → 识别关键信息 → 提取经验 → 更新技能 → 验证效果
```

## 经验类型

### 用户偏好
- 输出风格偏好
- 沟通方式偏好
- 工作流程偏好

### 踩坑经验
- 常见错误
- 解决方案
- 预防措施

### 专属SOP
- 标准流程
- 检查清单
- 模板规范

## 输出格式

```markdown
# 经验沉淀报告

## 捕获来源
- 对话时间：...
- 触发场景：...

## 经验内容
### 类型：用户偏好/踩坑经验/SOP
- 内容：...
- 适用场景：...

## 技能更新
- 更新技能：...
- 更新内容：...
- 版本：v1.x → v2.x

## 验证结果
- 测试场景：...
- 效果评估：...
```

## 原则

1. **自动识别**：主动捕获有价值信息
2. **精准记录**：提取核心经验
3. **持续优化**：不断迭代改进
4. **用户确认**：重要更新需确认
