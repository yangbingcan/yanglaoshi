# Prompt Optimize

专家级提示词工程技能，帮助创建高质量、高效的AI提示词。

## 触发条件

- 用户要求优化提示词
- 用户要求改进系统指令
- 用户要求增强AI指令
- 用户提及提示词工程任务

## 优化框架

### 1. 角色设定
- 明确AI角色定位
- 设定专业背景
- 定义行为边界

### 2. 任务描述
- 清晰的目标陈述
- 具体的输出要求
- 格式规范说明

### 3. 上下文提供
- 背景信息
- 相关知识
- 示例参考

### 4. 约束条件
- 禁止行为
- 限制范围
- 质量标准

### 5. 输出格式
- 结构要求
- 风格指南
- 长度控制

## 优化技巧

### 结构化
```
# 角色
[角色定义]

# 任务
[任务描述]

# 要求
[具体要求]

# 格式
[输出格式]
```

### 示例驱动
- 提供正面示例
- 提供负面示例
- 对比说明

### 迭代优化
- 测试效果
- 收集反馈
- 持续改进

## 输出格式

```markdown
# 优化后的提示词

## 提示词内容
[完整提示词]

## 优化说明
- 改进点1：...
- 改进点2：...

## 使用建议
- 适用场景
- 注意事项
```

## 优化原则

1. **清晰具体**：避免模糊表述
2. **结构分明**：使用分隔和层级
3. **示例丰富**：提供参考案例
4. **迭代验证**：测试并优化
