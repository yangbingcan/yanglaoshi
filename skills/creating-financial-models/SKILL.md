# Creating Financial Models

高级财务建模技能，支持估值分析、风险评估和投资决策。

## 触发条件

- 用户需要创建财务预测
- 用户需要估值模型
- 用户需要投资分析
- 用户需要财务建模

## 核心模型

### 1. DCF分析（现金流折现）
- 自由现金流预测
- 折现率计算（WACC）
- 终值估算
- 企业价值计算

### 2. 敏感性分析
- 关键变量识别
- 单因素敏感性
- 双因素敏感性
- 龙卷风图

### 3. 蒙特卡洛模拟
- 概率分布设定
- 随机变量生成
- 模拟运行
- 结果分析

### 4. 情景规划
- 基准情景
- 乐观情景
- 悲观情景
- 情景对比

## 输出格式

```markdown
# 财务模型报告

## 一、模型概述
- 分析对象
- 模型类型
- 关键假设

## 二、基础数据
- 历史财务数据
- 行业基准
- 增长假设

## 三、估值分析
- DCF估值结果
- 敏感性分析
- 情景分析

## 四、风险评估
- 关键风险因素
- 风险量化
- 缓解措施

## 五、结论与建议
- 估值区间
- 投资建议
- 注意事项
```

## 建模原则

1. **假设透明**：明确列出所有假设
2. **逻辑清晰**：公式可追溯
3. **敏感性检验**：关键变量敏感性
4. **保守估计**：避免过度乐观
