# Auto RedBook

小红书笔记素材创作技能，自动生成标题、正文和图片卡片。

## 触发条件

- 用户需要创建小红书笔记
- 用户需要小红书内容创作
- 用户需要生成图文卡片

## 内容创作规范

### 标题要求
- 20字以内
- 包含关键词
- 吸引眼球
- 可用表情符号

### 正文要求
- 300-800字
- 分段清晰
- 使用emoji
- 包含话题标签

### 图片卡片
- 封面图：吸睛设计
- 正文卡片：内容呈现
- 支持多种主题风格

## 笔记结构

```markdown
# [标题] ✨

[开场吸引]

📍 核心内容一
[内容]

📍 核心内容二
[内容]

📍 核心内容三
[内容]

[总结/行动号召]

#话题标签1 #话题标签2 #话题标签3
```

## 风格主题

### 1. 清新简约
- 浅色背景
- 简洁排版
- 柔和配色

### 2. 潮流时尚
- 鲜明对比
- 大胆配色
- 个性元素

### 3. 温馨治愈
- 暖色调
- 手绘元素
- 柔和氛围

## 输出格式

```markdown
## 笔记标题
[标题内容]

## 正文内容
[完整正文]

## 话题标签
#标签1 #标签2 #标签3

## 图片卡片
- 封面：[设计说明]
- 卡片1：[设计说明]
- 卡片2：[设计说明]
```

## 创作原则

1. **真实感**：避免广告感
2. **有价值**：提供实用信息
3. **互动性**：引导评论互动
